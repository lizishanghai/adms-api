## adms-api:
This is the rest api server for adms project. Built with Laravel 5.7.

## Get project
git clone https://github.com/lizishanghai/adms-api.git

## Config MySQL
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=blog
DB_USERNAME=blog
DB_PASSWORD=blog

## Start MySQL
sudo /usr/local/mysql/support-files/mysql.server start

## Start api server
php artisan serve
